# Калькулятор Кибер-рисков (СКФУ)

Интерактивный инструмент для оценки и приоритизации кибер-рисков на основе методологии ФСТЭК и принципа Парето.

## Особенности
- **Оценка угроз**: Анализ 227 актуальных угроз из БДУ ФСТЭК.
- **Матрица рисков**: Визуализация вероятности и ущерба.
- **Оптимизация Парето**: Выделение 20% критических факторов, вызывающих 80% последствий.
- **База вендоров и экспертов**: Проверенные решения и специалисты для минимизации рисков.
- **Экспорт отчетов**: Сохранение результатов в PDF.

## Технологический стек
- React
- TypeScript
- Tailwind CSS
- Lucide React (иконки)
- Framer Motion (анимации)
- Recharts (графики)

## Как запустить локально

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/imandritsa/kyberrisk.git
   ```
2. Перейдите в папку проекта:
   ```bash
   cd kyberrisk
   ```
3. Установите зависимости:
   ```bash
   npm install
   ```
4. Запустите сервер для разработки:
   ```bash
   npm run dev
   ```
5. Откройте `http://localhost:3000` в браузере.
